## 如何运行

1. 确认你的环境已经搭建完成: [搭建开发环境](https://reactnative.dev/docs/environment-setup)
2. 克隆这个仓库, 并且进入这个项目的根路径
3. 运行 `yarn`
4. 运行 `yarn android` 或者 `yarn ios`
